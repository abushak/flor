
-- SUMMARY --

The Views Details module adds details row plugin to views.

For a full description of the module, visit the project page:
  http://drupal.org/node/2065703

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/2065703


-- REQUIREMENTS --

views.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- USAGE --

* While creating view choose display format details. OR Update the display
  format of the existing view.

* After saving the display format choose the settings for details.
