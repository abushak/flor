
-- SUMMARY --

The Views Fieldset module adds fieldset row plugin to views.

For a full description of the module, visit the project page:
  http://drupal.org/node/1867574

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/1867574


-- REQUIREMENTS --

views.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- USAGE --

* While creating view choose display format fieldset. OR Update the display
  format of the existing view.

* After saving the display format choose the settings for fieldset.
